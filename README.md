## Description üì±

The **YouTubeDownloader** module allows you to download videos from YouTube in various qualities. You can select the desired resolution and receive the video directly in the chat.

### Commands

- `.ytdl <video_url>` ‚Äî Download a video from YouTube.

### Usage Example

1. Type `.ytdl https://www.youtube.com/watch?v=dQw4w9WgXcQ` to start downloading.  
2. Choose the quality from the available options (e.g., 720p).  
3. Once processed, the video will be sent to the chat.
